# Al-Daleel Medical Laboratories Website

Complete responsive website for Al-Daleel Medical Labs, built with React, Vite, and Tailwind CSS.

## Features
- **RTL Support**: Fully Arabic interface with `dir="rtl"`.
- **Responsive Design**: Mobile-friendly layout for all pages.
- **Pages**: Home, About, Services, Test Menu, Booking, Results Portal, Contact, Physicians Portal.
- **Searchable Test Menu**: Filter by category and search by name.
- **Booking Form**: With validation and success simulation.
- **SEO Optimized**: Meta tags, JSON-LD Schema, Sitemap, Robots.txt.

## Tech Stack
- React 19
- Vite
- Tailwind CSS 3
- React Router DOM
- React Helmet Async (SEO)
- Lucide React (Icons)

## Installation & Running

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Project Structure
- `src/components`: Reusable UI and Layout components.
- `src/pages`: Main page components.
- `src/data`: Static data and schema.
- `public`: Static assets (robots.txt, sitemap.xml).
